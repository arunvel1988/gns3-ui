Network Playground

Welcome to your Network Engineering Lab. This environment helps you learn and experiment with modern networking concepts, tools, and architectures.

  • Learn routing and switching concepts using VyOS and Linux networking
  • Explore NFV (Network Function Virtualization) and build virtual routers, firewalls, and load balancers
  • Practice SDN (Software Defined Networking) using Open vSwitch (OVS) and OpenDaylight
  • Configure firewalls like Palo Alto and FortiGate for LAN, DMZ, and Internet zones
  • Simulate data center or enterprise topologies using GNS3, VMware, and Linux bridges
  • Test automation using Ansible, Terraform, and Python for network provisioning
  • Understand network monitoring using Grafana, Prometheus, and SNMP tools
